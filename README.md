# Planning System

A Windows desktop task planning application built with Tauri + React + TypeScript + Supabase.

## Features

- Neon blue/green dark theme using OKLCH color space
- Task creation with 6-step flow
- Task persistence to Supabase
- Daily task limits: Agentic (unlimited), Hybrid (4/day), Non-Agentic (3/day)
- Actor management with notes
- Trajectory editing
- Copy/Sent buttons with feedback
- Mobile-responsive design

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS with OKLCH color tokens
- **Backend**: Supabase (PostgreSQL + Realtime)
- **Desktop**: Tauri
- **State**: Zustand

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
